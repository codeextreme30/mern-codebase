# MERN Codebase

## Project Structure

The actual project code is in different branches:

- **develop** - Backend code (inside ackend/ folder)
- **main** - Empty (reserved for future releases)

## Get Started

```bash
git checkout develop
cd backend
npm install
npm run dev
```

## Branches

- develop - Active development branch with backend code
- main - Protected main branch

For full documentation, switch to the develop branch.
